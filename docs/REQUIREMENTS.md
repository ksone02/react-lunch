# 점심 뭐먹지 기능 목록

## components 
- Header
    - '점심 뭐먹지'를 누르면 선택 옵션이 초기화된다. 

- SelectBox
    - 카테고리 필터 기능
    - 이름 및 거리 정렬기능

- RestaurantItem

- RestaurantList
    - 새로고침해도 추가한 음식점 정보 유지된다. -> local storage

- BottomSheet
    - 음식점을 누르면 음식점 상세 바텀시트가 열린다.
    - 닫기 버튼 혹은 바텀시트 백드롭을 누르면 바텀시트가 닫힌다. 

- RestaurantDetail

- Button
